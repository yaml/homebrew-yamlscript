homebrew-yamlscript
===================

[Homebrew](https://brew.sh/) for YAMLScript


## Install YAMLScript with Homebrew

Works on Linux Intel and macOS ARM.

```
$ brew install yaml/yamlscript/ys
$ brew install yaml/yamlscript/ys@0.2.19
$ brew install yaml/yamlscript/libys
$ brew install yaml/yamlscript/libys@0.2.19
```
